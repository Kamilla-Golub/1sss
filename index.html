Как работает 1С: Кабинет сотрудника

Руководитель Со стороны работодателей

Не придется реорганизовывать процесс обработки, подписания и отправки документов при переходе на систему КЭДО. Использование привычного интерфейса программы 1С значительно облегчает этот переход и не потребует дополнительного обучения.

Для подписи документов со стороны работодателя используется усиленная квалифицированная электронная подпись (УКЭП).

Со стороны сотрудников


Удобный способ для сотрудников отправлять заявления, запросы и подписывать кадровые документы. Личный кабинет, доступный через мобильное приложение или веб-браузер, это предоставляет сотрудникам гибкость и возможность управлять своими кадровыми процессами с любого устройства.

Для подписи документов со стороны сотрудников используется усиленная неквалифицированная электронная подпись (УНЭП)

Чем поможет переход на КЭДО?

Сотруднику:

С помощью приложения сотрудник может:

  1. Получить расчетный лист.

  2. Посмотреть свой запланированный
    отпуск по графику.

  3. Запросить справки: 2-НДФЛ, справку с места работы,
    справки по неполным отпускам.

  4. Подать заявление на отпуск.

  5. Согласовать отсутствие.

  6. Увидеть отсутствие коллег.

  7. Подать заявление на изменение личных данных.

Руководителю

Для руководителя есть следующий функционал:

  1. Согласование заявлений сотрудников.

  2. График отпусков.

  3. Список сотрудников, не отметивших
    ознакомление с расчетным листом.

Бухгалтеру-кадровику

Функционал сервиса позволяет:

  1. Публиковать расчетные листы до подключения сервисов.

  2. Публиковать расчетные листы по разнным направлениям.

  3. Видеть список заявок сотрудников в разделе «Мои дела».
